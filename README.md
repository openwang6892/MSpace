## 功能
1. 可计算使用天数
2. 彩虹username
3. ffmpeg工具箱
4. APT管理
5. termux功能菜单
6. 系统信息
7. 随机龙图
8. JS题库
9. 终端使用Shizuku
10. 新闻
11. 多种资源

任你使用

## 其他
MSpace仅支持bash，请勿使用其他解释器运行
建议只在Termux运行MSp
